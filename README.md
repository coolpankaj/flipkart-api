# flipkart-api
flipkart RESt api using ExpressJS

DEMO : https://pankajsaini123.github.io/flipkart-api/

# Functionalities

- Add item publically
- Add item to cart
- Delete product
- Delete product from cart
- Edit product by itemId
- Get all products
